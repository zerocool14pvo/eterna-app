# app

Репозиторий приложения **Eterna** (React + Vite + TypeScript). Исходный код и сборка проекта — в этом репозитории.

## Скрипты

- `yarn dev` — разработка
- `yarn build` — production-сборка
- `yarn lint` / `yarn format` — проверки и форматирование

Подробные правила для разработки и ИИ — в [`AGENTS.md`](./AGENTS.md).
